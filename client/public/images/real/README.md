**How to add:**
1. Save or drag-and-drop your photos into this folder
2. Rename them exactly as shown above
3. The website will pick them up automatically — no code changes needed

**File size tips:**
- Keep each photo under 500 KB for fast loading
- JPG quality 80-85% is plenty for web
- If a photo is bigger, you can optionally run: `sips -Z 1920 filename.jpg` in Terminal to resize
